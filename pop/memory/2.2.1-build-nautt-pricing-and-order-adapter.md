# 2.2.1-build-nautt-pricing-and-order-adapter — Build Nautt pricing and order adapter

Delivered a server-only Nautt pricing and order boundary on `develop`. Public DTOs remain bounded and redacted, with optional PIX copy-and-paste normalization and no hosted-link path.
