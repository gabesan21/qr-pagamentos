## Abort conditions

- Any operation would delete, overwrite, relocate, or reformat unrelated PoP harness or user-owned files.
- No stable Next.js 16 release supports Node 24 LTS, or a concrete target host constraint rejects Node 24.
- A required dependency resolves only to prerelease/canary or reports a known critical vulnerability with no compatible stable patch.
- The scaffold requires database, Docker, authentication, provider credentials, or a product/UI decision outside this task to pass its baseline checks.
- A changed path cannot be explained by the root DOX contract and an objective local contract cannot be written within the task scope.
