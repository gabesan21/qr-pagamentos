# 4.3.1-owner-and-admin-order-views — Owner and admin order views

- **Phase:** [[pop/roadmap/4-checkout-and-orders|Phase 4.3]]
- **Plano:** [[4.3.1-owner-and-admin-order-views.plan]] · **Aprovação:** [[4.3.1-owner-and-admin-order-views.approval]] · **Verificação:** [[4.3.1-owner-and-admin-order-views.verify]]

## O quê

Add owner-scoped and administrator order views with protected customer-data access: each owner lists and inspects only their own payment-link orders, administrators retain global operational visibility, and customer data collected under each account's checkout policy is exposed only to the authorized principal.

## Por quê

It closes Phase 4.3 and Epoch 4: orders created by the public checkout currently have no protected visibility surface, blocking storefront/production work that depends on order operations.
