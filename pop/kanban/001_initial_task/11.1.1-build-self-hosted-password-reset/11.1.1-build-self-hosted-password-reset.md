## What

Add validated file-backed mail configuration and canonical public origin plus generic request/consume flows using hashed single-use expiring reset challenges, abuse limits, password rotation, session revocation, bilingual mail, and redacted evidence.

## Why

Administrators must be able to send a reset email, but the project currently has no transport or token boundary and email must remain optional and non-login.
